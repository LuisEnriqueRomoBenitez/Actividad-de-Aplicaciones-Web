HTML 1.0

Description

First released in 1993, before it, not many developers were creating websites, so it was evolving slowly during those time. This version only had basic tags like text, even tables, and fonts were not available in this version

    Features

  • Static pages for the user's visitors
  • The use of frames or framesets
  • Own HTML extensions such as blinking and marquees, labels introduced during the browser wars
  • Guestbooks
  • GIF buttons
  • HTML forms sent via e-mail

      • Advantages

  • A great field of opportunities for developers
  • Innovation opportunity
  • Exposure to the whole world through the internet
  • The developers of the information has full control and authority of what is published

      • Disadvantages

  • His technologhy is associated with a slow engine
  • Passive communication
  • It cannot permit save the information adn this means that we can lose the new information posted
  • It doesn't have an inmmediately reaction between public and the information that has been sharing

    • HTML 2.0

    Description

    It was publushed in 1995. All features of HTML 1.0 were inherited in HTML 2.0 but with new plus features were added

      Features

  • Structural elements for paragraphs, headings, ordered and unordered lists
  • Defined menu and dir
  • Web as a platform

      • Advantages

  • Provide a level user interaction that was not available before
  • Websites have become much more dynamic for develop code
  • Marking easier share the message or the purpose of a page

      • Disadvantages

  • Tools to be developed
  • Difficult to recognize the structure
  • Difficult to identify for someone who's not a developer

    • HTML 3.O

    Description

    It remains the standard markup language for creating websites

      Features

  • Includes support for customized lists
  • Fine positioning control with entities like &empspace
  • Horizontal tabs and horizontal aligment of headers and paragraph text
  • Static area banner for corporate logos, disclaimers and customized navigation

      • Advantages

  • Show multimedia as images and animations
  • Multi-support distribution
  • Variety of content, not just a page with letters

      • Disadvantages

  • Still being a slow design
  • Still having a static design
  • Limited tags

    • HTML 4.01

    Description

    Published as recommendation for W3C in 1999. This version was the most successful of all previous HTML released versions, this version features like multimedia, scripting, better printing features and more were added. After HTML 4.01, no newer version of HTML was released for many years because W3C's HTML working group were busy building the language XHTML

      Features

  • More multimedia options
  • Scripting languages
  • Style sheets
  • Better printing facilities
  • Documents more accesible for users with disabilities

      • Advantages

    • Allow multiple documents withing a single Web page
    • Possible to load page form different servers in a single frameset
    • This version is the longest version ever created because, it was a complete language with his elements, content, and new attribute that we can use in our design

        • Disadvantages

    • Frames can make the production of a website complicated
    • Search engine might link directly to a page that was intended to be displayed within a frameset
    • Users have become so familiar with some developers and code terms for anavigate and identify a structure of a page

      • HTML 5.0

      Description

      Released in 2014,extended version of HTML, extended version of HTML of 4.01 published in 2012, many new tags were added in this version like *header, footer, main video, etc)

        Features

    • New semantic elements: header, footer and section
    • We can achieve storage without use third-party plugins
    • We can add multimedia as: change the border, opacity, reflections, gradients, transitions, transformations, and even animations
    • Visitors can decide if they want to share their location in realtime
    • Microdata
    • Drag and drop
    • Nav tag
    • Placeholder attribute
    • Progress tag
    • Figure and fig caption
    • Vector graphics
    • Email attribute

        • Advantages

    • Provides audio and video support
    • Clear and consistent code
    • Web sites more consistence
    • More page layout elements available for your content
    • Search engine optimization
    • Storage options are more reliable
    • All compatible browsers collect and use data
    • Modern user experience

        • Disadvantages

    • There are different video supports for HTML5
    • It requires modern browser to access it
    • There are media licensing issues which must be considered
    • Multiple device responsiveness can be a headache
    • Is always updated
